## 当前活跃问题

1. **DeepSeek 生成代码偶发语法误报**：生成的代码块中若包含 Windows 路径裸反斜杠（如 `C:\Users\...`），AST 解析会报 `unexpected character after line continuation character`。建议在 Prompt 中强制使用原始字符串或正斜杠路径，或增加代码修复后处理。
2. **前端 chunk 体积大**：mermaid + pyodide 导致部分 chunk >500KB。后续可通过动态导入 (`import()`) 或 `rollupOptions.output.manualChunks` 拆分。
3. **API key 明文存储**：`.env` 中仍为明文，已确认 `.gitignore` 包含 `.env`。
4. **Windows 终端中文乱码**：部分测试输出显示乱码，属 PowerShell/GBK 编码问题，不影响实际运行。
