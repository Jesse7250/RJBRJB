## 本轮完成工作

### ✅ 1. 资源生成 SSE 异步化 + 进度推送
- 后端新增 `GET /api/resources/stream-generate`（SSE）。
- `AgentOrchestrator.generate_resource_stream` 为异步生成器，分阶段 yield 进度事件：
  - `builder`：正在生成教学资源
  - `validation`：神经符号校验
  - `debate`：Society of Mind 辩论议会
  - `complete`：最终结果
- 同步 LLM 调用使用 `asyncio.to_thread` 避免阻塞事件循环。
- 保留旧同步接口兼容测试。

### ✅ 2. ChatPanel SSE 流式输出
- 后端新增 `GET /api/sessions/{session_id}/chat-stream`。
- 前端 `ChatPanel` 改为使用 `fetch + ReadableStream` 手动解析 SSE：
  - 显示"正在理解你的意图..."
  - 收到 complete 后展示 Agent 名称和回复内容。
  - 出错时 fallback 到错误提示。
- 消息气泡现在显示 Agent 头像/名称。

### ✅ 3. 后端代码执行与判题接口
- 新增 `app/services/code_executor.py`：
  - AST 静态安全检查
  - 标准库导入白名单
  - 禁止 `open/exec/eval/os.system` 等危险调用
  - subprocess 隔离执行 + 5 秒超时
- 新增 `app/api/code.py`：
  - `POST /api/code/execute`：执行 Python 代码
  - `POST /api/code/judge`：执行并对比期望输出
  - `POST /api/code/judge-exercise`：判题并预留学习日志入口
- 在 `main.py` 注册 `/api/code` 路由。
