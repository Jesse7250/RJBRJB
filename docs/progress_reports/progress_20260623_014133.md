## 当前活跃问题

1. **DeepSeek 生成耗时波动**：30-70s 不等，受网络/API 负载影响。缓存可在重复请求时显著改善，但首次仍需等待。
2. **内存缓存非持久化**：进程重启后缓存清空。后续可接入 Redis 或 SQLite 持久化缓存。
3. **API key 明文存储**：`.env` 中仍为明文，已确认 `.gitignore` 包含 `.env`。
4. **Windows 终端中文乱码**：部分 PowerShell 输出显示乱码，属编码问题，不影响实际运行。
